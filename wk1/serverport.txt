1. What is the definition of server port?
A server port is a logical communication endpoint used by a computer to identify specific processes or services on a network. It allows many services (like web, email, FTP) to run on a single device using different port numbers.

2. What is server port range?
The server port range is from 0 to 65535. These ports are divided into:
- Well-known ports: 0–1023
- Registered ports: 1024–49151
- Dynamic/Private ports: 49152–65535

3. How many bits does number above represent?
The port number range (0–65535) represents 16 bits, since 2^16 = 65536 possible values.
